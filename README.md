# 2048-Game
### This game is called 2048.
#### The game contains a grid with 4 rows and 4 columns.
#### Initially, the grid will have two 2's in random tiles.
#### For example:
#####    0  0  0  0
#####    0  2  0  0
#####    0  0  0  2
#####    0  0  0  0
#### These tiles can be moved in any direction using the controls:
#####    w: Up
#####    a: Left
#####    s: Down
#####    d: Right
